Air quality: World Air Quality Index (WAQI) project, CPCB Bengaluru stations. Weather: Open-Meteo. AQI forecasts are produced by a statistical time-series model where enough history exists, WAQI's own outlook where it doesn't, or a flat estimate as a last resort — each chart says which. Government monitoring stations occasionally stop reporting for extended periods; when that happens the station's "last observed" time says so. Weather forecasts are Open-Meteo's own numerical model, used as-is. Pipeline runs automatically every hour via GitHub Actions.
